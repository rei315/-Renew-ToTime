# -Renew-ToTime
Renewed Version of ToTime Application
IOS Application that alerts you when you reach the distance you have set

# Installation
pod install

## Using
- Architecture: MVVM

- Library : 
1. RxSwift
2. SnapKit
3. CoreLocation
4. RealmSwift
5. CoreAnimation
6. Alamofire -> Google API (Reverse-Geocode, Place-Detail, AutoComplete)
